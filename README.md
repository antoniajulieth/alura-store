# Desafío Alura Store
Este proyecto se realizó como parte del Desafío Alura Latam de Ciencia de Datos. El objetivo es ayudar al Sr. Juan a decidir qué tienda de su cadena Alura Store debería vender para poder iniciar un nuevo negocio. Para tomar esta decisión, se analizan los datos de ventas, métricas de rendimiento y opiniones de los clientes de las cuatro sucursales. Con base en este análisis, se busca identificar cuál tienda tiene el menor desempeño y presentar una recomendación basada en datos.

# Descripción
## 1. Análisis de los datos de las tiendas

Se evalúan distintos indicadores del desempeño de cada tienda, entre ellos:

Ingresos totales

Categorías de productos más y menos vendidas

Calificaciones y reseñas de los clientes

Productos con mayor y menor volumen de ventas

Tiempo promedio de entrega

Mapa de calor de ventas

## 2. Creación de visualizaciones

Se utilizan diferentes tipos de gráficos para presentar los resultados de manera clara y comprensible. Entre ellos se incluyen:

Gráficos de barras

Gráficos circulares

Mapas de calor

Gráficos de dispersión

Gráficos de violín

## 3. Recomendación final

A partir del análisis realizado, se elabora una conclusión que responde a:

¿Qué tienda debería vender el Sr. Juan?

¿Qué justifican esa decisión según los datos?
